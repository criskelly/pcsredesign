About This Site

Redesign

Over the past year, Portland Code School has experienced an extraordinary rate of change and growth. To celebrate and encapsulate the new experience PCS provides, a complete website redesign was in order!

Tools:

This site is built using Middleman, GitHub Pages, the Middleman GitHub Pages gem, Font Awesome by Dave Gandy, and Bootstrap 3.

Core Development Team:

  • Technical and Design Lead: Shawna C. Scott
  • Timothy Winters

Additional Contributors:

  • Al Zimmerman
  • Cris Kelly
  • Chris Peddecord

Contributing:

Contributors are always welcome! If you would like to contribute, please read the documentation for any of the tools we use you are not familiar with, listed above. Once you've done that, there are three different ways you can contribute:

  1. File an issue if you find any problems, or have suggestions for improvements or additional content you'd like to see.
  2. For those of you new to the GitHub contribution game, check out our open issues to get an idea of what to work on.
  3. If you have a feature you'd like to see on our site and want to tackle it yourself, feel free to fork our repo and send a pull request to our integration branch.

    4. NOTE: Keep in mind that we may not merge every pull request if it doesn't fit with our overall mission and design plan. It is not a reflection on you or the quality of your code!
