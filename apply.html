Apply To Portland Code School!

Thank you for your interest in attending Portland Code School! Please be sure to review our classes page to decide which of our courses best fits your needs. If you aren't sure which class is for you, feel free to email us for more information.

We look for students who will work hard, challenge themselves, and be team players. This program is accelerated, and you will be required to put in time outside of class for learning, coding, and pairing if you want to succeed. Part of this application process is showing that you can learn new things and overcome technical challenges. Don't be intimidated if we ask you to do something you aren't familiar with yet! Use the linked resources we've provided, and you should be just fine. Show us you would be a stellar student at PCS!

Are You Ready?

Note: If you have no programming experience, we highly encourage you to apply for and attend our Primer course before applying to either the Ruby on Rails or Full-Stack JavaScript course.

All prospective students need to have these materials ready to complete the application:

  1. A GitHub account, for creating Gists.
  2. A Gist of your resume. Bonus points for formatting it with Markdown.
  3. A Gist of your cover letter. Describe your interest in programming, any experience you have coding, and how you function in a team work environment. Again, we love Markdown!
  4. Any optional extras or class-specific requirements.

Extras

Ruby on Rails
  • The link to a GitHub repository containing a "Hello World" app in Ruby on Rails. The first chapter of Michael Hartl's Ruby on Rails Tutorial will walk you through the process.
Full-Stack JavaScript
  • The link to a GitHub repository containing an index.html page. The index page can simply say "Hello World" on it.
Optional
  • Links to any code samples or portfolio items you are working on.
  • Digital evidence of you being awesome. A video of you rapping, a poem about trees, or an animated gif of you doing interpretive dance
  • Description of any tech community involvement, including attendance at local user groups; activity on any mailing lists, forums, or IRC channels; an active Twitter account; etc. Calagator is a great resource for finding ways to get involved!
