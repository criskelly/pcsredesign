Expectations

What's the minimum skill level for the program?

This will depend on which track you enter.

For the JavaScript track, you should be able to:

  • Explain front-end vs. back-end web development
  • Draw a client-server model diagram
  • Open files in Sublime Text 2
  • Push new or altered files to a Github repo
  • Create a new file in a specific directory from the command line

For the Ruby on Rails track, you should be able to:

  • Demonstrate understanding of programming basics
  • Do things in Interactive Ruby (irb)
  • Show us your Github account with self-created repos and lots of "green"
  • Write simple functions in Ruby, require gems, and generate other simple code in Ruby
  • Use the command line, Sublime Text 2, and Github with relative ease

How can I do better in the application process?

There are three things you can do.

  • Creative Project: Make a video or other creative project that features you doing whatever it is you do to help us get to know you better.
  • Portfolio: Make a portfolio of your previous learning attempts. We aren't expecting something like worrydream.com, but it's a great example of a superior portfolio piece.
  • Community Engagement: Get involved in a programmer community such as Code Scouts, Ruby Newbies, Hack + Help, RailsGirls/RubyNest, JavaScript Admirers, or any of a hundred other options you can find on Calagator.

Can I really learn to code in 12 weeks?

Yes, but you should expect to work hard in this program (about 2-3 hours outside of class for each hour in class). Our programs each meet two nights weekly (3 hours each night), and one Saturday each month (6 hours more). We want to make sure you are prepared for a career as web developer, which means there is a lot of ground to cover.

Will we be guaranteed a job after the program?

No, there is no guarantee. But if you do well we will try our hardest to make sure you find your way into the loving arms of a great developer shop.

What if I can't pay?

We endeavor to not turn away anyone for financial reasons. Let us know if tuition presents a significant financial difficulty and we will do everything we can to work with you.

Do I need to be enrolled in college to take part?

Not at all. This is a private career school.

Will we be working alone or in teams?

We believe strongly in pairing and working in small groups. There will be some individual work time and some larger group work time, but you will spend most of your time pairing with another student (or two) or a mentor.

Will there be any open source work?

Yes. We encourage open source work and provide time in the final phase of the program for guided open source work.

Teaching details

What happens if I can't keep up or if I find out that I hate programming?

This program is going to be hard and the only way to know how far you can go is to try. We promise to do everything we can to help you through the program and provide extra resources and help sessions for you.
But, at the end of the day if you need to drop the program, we offer a pro-rated tuition refund. The pro-rated refund does not include the $1,000 deposit.
